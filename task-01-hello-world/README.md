# Task 01 — Hello, World!

Your very first Fortran program. The tradition is sacred. Do not skip it.

## Task

Write a program that prints the following line to the screen:

```
Hello, World!
```

## What to learn

- Basic structure of a Fortran program (`PROGRAM` / `END PROGRAM`)
- Printing output with `PRINT *`

## Compile & Run

```bash
gfortran solution.f90 -o solution
./solution
```
